# LockdownMode
[![Build status](https://ci.appveyor.com/api/projects/status/m1swbs685ntqw01w?svg=true)](https://ci.appveyor.com/project/coreizer/lockdownmode)

LockdownModeはマウスやキーボード類のデバイスを一時的に無効化する機能を提供します。

### 機能
- キーボードの入力阻止
- マウスのカーソルを真ん中に固定する
